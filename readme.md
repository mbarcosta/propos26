# propos26

Repositório principal do projeto **propos26**, voltado ao desenvolvimento de serviços e sistemas de automação inteligente  de  processos de negócio do domínio da pós-graduacão. 

## Objetivo

O projeto tem como objetivo integrar serviços independentes (microservices) para apoiar automação de processos, incluindo:

- leitura e processamento de e-mails
- tomada de decisão baseada em regras
- integração com motores de processo (ex: Camunda)
- serviços auxiliares (ex: validação de CPF)
